NEU: Stockfish mit NNUE (Efficiently Updatable Neural Network).

Um eine der folgenden Engines verwenden zu können, benötigen Sie eine Schachanwendung, die mit dem Open Exchange-Protokoll kompatibel ist.

Dieses Paket enthält die folgenden Open Source-Engines:

Stockfish 11

Stockfish 20200907

Stockfish 20200907 (nn-308d71810dff.nnue)

Stockfish wird entwickelt von Tord Romstad, Marco Costalba, Gary Linscott und Joona Kiiiski.
Die Engines werden von Peter Österlund und Bernhard C. März zusammengestellt.
Stockfish Engines OEX wird von Karl Schreiner entwickelt.

Stockfish Engines OEX Sammlung wird unter den Bedingungen der GNU General Public License v3 veröffentlicht (GNU GPL v3). Der Quellcode ist auf  GitHub  verfügbar.
