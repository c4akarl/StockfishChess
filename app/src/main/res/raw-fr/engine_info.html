Mise à jour: Stockfish 14 pour les processeurs arm64-v8a et armeabi-v7a.

Ce package fournit les moteurs open source suivants:

Stockfish 12

Stockfish 13

Stockfish 14

Pour utiliser ces moteurs, vous avez besoin d'une application d'échecs compatible avec le protocole Open Exchange.
GUI d'échecs recommandées:

* Chess for All
* DroidFish
* Scid on the go
* Chess PGN Master
* Chess for Android
* Chess (Jeroen Carolus)

Stockfish est développé par Tord Romstad, Marco Costalba, Gary Linscott et Joona Kiiiski.
