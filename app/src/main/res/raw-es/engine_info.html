Actualización: Stockfish 12, ahora también para procesadores armeabi-v7a.

Este paquete proporciona los siguientes motores de código abierto:

Stockfish 11

Stockfish 12

Stockfish 20200926

Stockfish 20200926 NNUE

Para utilizar estos motores, necesita una aplicación de ajedrez que sea compatible con el protocolo Open Exchange.
GUI de ajedrez recomendadas:

* Chess for All
* DroidFish
* Scid on the go
* Chess PGN Master
* Chess for Android
* Chess (Jeroen Carolus)
* Acid Ape Chess

Los motores son compilados por Peter Österlund y Bernhard C. März.
