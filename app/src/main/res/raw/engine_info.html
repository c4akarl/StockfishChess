NEW: Stockfish with NNUE (Efficiently Updatable Neural Network).

To use any of the following engines, you need a chess application compatible with the Open Exchange protocol.

This package provides the following open source engines:

Stockfish 11

Stockfish 20200907

Stockfish 20200907 (nn-308d71810dff.nnue)
